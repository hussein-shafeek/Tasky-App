# Tasky App 

Tasky App is a simple, intuitive, and powerful Todo/Task management app that allows users to create, edit, and delete tasks, set priorities, track status, and manage their workflow efficiently.


## Features 

- Add new tasks with title, description, and optional image  
- Edit and update existing tasks  
- Delete tasks  
- Set task priority: Low, Medium, High  
- Track task status: Waiting, In Progress, Finished  
- Assign due dates to tasks  
- Filter and sort tasks by priority, status, or due date  
- Search tasks by title or description  
- User authentication: Login and Register  
- JWT-based secure API authentication  
- Responsive and user-friendly UI  
- Demo video available to see the app in action  


## Requirements 

- Flutter >= 3.0  
- Dart >= 3.0  
- Android Studio or VS Code with Flutter setup  
- An Android/iOS device or emulator  


## Usage 

- Open the app and click **Add Task** to create a new task.  
- Tap any task to edit or delete it.  
- Filter or sort tasks by priority, status, or due date.  
- Search tasks by typing keywords in the search bar.  
- Login/Register to sync your tasks securely with the backend.
